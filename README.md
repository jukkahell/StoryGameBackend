## Description

Sanapeli mobile app's backend service

## Installation

```bash
$ npm run install:all

# Set environment
$ set/export NODE_ENV=development
```

## Running the app

```bash
# run db migrations if needed
$ npm run migrate up

# development
$ npm run start

# watch mode
$ npm run start:dev

# incremental rebuild (webpack)
$ npm run webpack
$ npm run start:hmr

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
